## Features

**Audio Playback**
- Built-in procedural synthesizer tracks — works instantly, no files needed
- Local MP3 file upload support via drag-and-drop or file picker
- External audio URL streaming with automatic HTTPS enforcement
- Full transport controls: Play, Pause, Stop

**Visual Experience**
- Three real-time visualizer modes: Retro Bars, Sine Wave, Cyber Pulse
- Customizable LED accent color with live preview
- Animated cassette spindles that spin during playback
- VU meter display synced to audio output
- Fullscreen background canvas visualization

**User Interface**
- Editable playlist with track type badges
- Click-to-select track loading with auto-play
- Retro-styled deck panel with digital time display
- Responsive layout that adapts to screen size

## How to Use

**Option 1: GitHub Pages (Recommended)**
1. Fork this repository
2. Go to Settings > Pages
3. Select your main branch as the source
4. Access your live deployment at `https://yourusername.github.io/repository-name`


**Playing Music**
- Click any built-in synth track from the playlist to start immediately
- Use the file upload button to play your own MP3 files
- Paste an HTTPS audio URL and click "Add URL Track"
- Generate additional procedural tracks with the synth button

## Technical Details

**Core Technologies**
- Web Audio API for real-time analysis and synthesis
- Canvas API for background visualization rendering
- CSS custom properties for dynamic theming
- Vanilla JavaScript with zero external dependencies
